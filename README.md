# Spark & Gather | Corporate Event Management 🚀

A modern, responsive multi-page website for a boutique event management agency. This project showcases frontend development skills using **Bootstrap 5**, custom **CSS3 animations**, and the **Intersection Observer API** for scroll-triggered effects.

## ✨ Key Features
- **Dynamic Scroll Animations:** Custom JavaScript triggers for smooth entry effects.
- **Responsive Design:** Mobile-first approach using Bootstrap's grid system.
- **Interactive Services UI:** A "Show More" toggle system for a clean service catalog.
- **Form Integration:** Newsletter subscription handled via Web3Forms API.
- **Clean Branding:** Professional typography and a high-contrast "Spark & Gather" color palette.

## 🛠️ Tech Stack
- **HTML5 & CSS3** (Custom animations & linear gradients)
- **Bootstrap 5.3** (UI Components & Layout)
- **JavaScript** (Intersection Observer, Form handling, Toggle logic)
- **Animate.css** (Hero section entrance)

## 📂 Project Structure
- `index.html`: Main landing page with hero, intro, and service overview.
- `services.html`: Detailed service catalog with interactive toggle.
- `style.css`: Custom overrides for branding and "Glassmorphism" navbar effects.
